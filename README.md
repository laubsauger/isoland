# isoland
[![Build Status](https://travis-ci.org/laubsauger/isoland.svg?branch=master)](https://travis-ci.org/laubsauger/isoland)

canvas based isometric engine

## Features
### Done
- basic rendering
    - iso renderer
    - zoomable minimap (2d renderer)
    
- basic input processing
    - mouse hover

### Todo
